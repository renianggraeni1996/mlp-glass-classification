Implementasi Multilayer Perceptron (MLP) untuk klasifikasi jenis kaca menggunakan dataset Glass Identification dari UCI Machine Learning Repository.

Glass Identification Dataset
https://archive.ics.uci.edu/dataset/42/glass+identification

MLPClassifier dengan variasi hidden layer:
- hidden_layer_sizes=(50,)
- hidden_layer_sizes=(50,)
- hidden_layer_sizes=(100,)
- hidden_layer_sizes=(100,50)

Metode evaluasi:
- Accuracy
- Precision
- Recall
- F1-score
